프로젝트명: BNI Korea Leadership Training 2025 - Google Sheets 연동 트레이닝 신청 사이트

프로젝트 개요: 기존 아임웹 기반 BNI Korea Leadership Training 사이트(https://www.ltt-bnikorea.com/)를 React 기반으로 재제작하며, 구글 스프레드시트와의 실시간 연동 기능을 추가하여 트레이닝 일정, 결제 링크, 공지사항을 자동으로 관리할 수 있는 시스템 구축

핵심 기능 요구사항:

1. 사이트 구조 및 디자인
메인 페이지: "2025 BNI Korea Leadership Training" 타이틀과 소개 문구
네비게이션 메뉴: Program, Classroom, BNI 코리아 스토어 바로가기 버튼
반응형 디자인: 모바일, 태블릿, 데스크톱 최적화
다크 테마: 기존 사이트와 동일한 검은색 배경 기반 디자인
BNI 브랜딩: BNI 로고 및 브랜드 컬러 적용
2. 구글 스프레드시트 연동 시스템
실시간 데이터 연동: Google Sheets API를 통한 실시간 데이터 fetch
자동 업데이트: 스프레드시트 변경 시 웹사이트 자동 반영
다중 시트 지원: 여러 트레이닝 프로그램별 시트 관리
3. 트레이닝 프로그램 관리
프로그램 목록 표시:

파운데이션 트레이닝
멘토링 코디네이터 트레이닝
PR 코디네이터 트레이닝
ST & 도어퍼슨 트레이닝
이벤트 코디네이터 트레이닝
멤버십 위원회 트레이닝
교육 코디네이터 트레이닝
비지터 호스트 트레이닝
프로그램별 정보 표시:

트레이닝 제목
일정 및 시간
장소 (오프라인/온라인)
트레이너 정보
신청 현황
결제 링크
상태 (예정/진행중/종료)
4. 동적 콘텐츠 관리
일정 관리:

신규 일정 자동 추가
일정 변경 시 실시간 반영
종료된 일정 자동 표시 변경
공지사항 관리:

중요 공지 상단 고정 표시
일반 공지 목록 표시
공지 우선순위 설정
강의 다시보기 링크:

종료된 강의의 녹화본 링크 자동 게시
수강 가능 기간 표시
접근 권한 관리
5. 결제 및 신청 시스템
신청 폼:

개인정보 입력 (이름, 연락처, 이메일, 소속 챕터)
참여 희망 트레이닝 선택
실강/녹화본 선택 옵션
결제 연동:

결제 링크 자동 생성 및 관리
결제 현황 추적
결제 완료 시 자동 확인
6. 관리자 기능
스프레드시트 템플릿:

표준화된 데이터 입력 형식
데이터 검증 규칙
자동 계산 필드
상태 관리:

트레이닝 상태 (예정/진행/종료)

트레이닝명으로 검색
날짜별 필터링

알림 시스템:

신규 일정 알림
일정 변경 알림
신청 마감 임박 알림
8. 기술 스택 및 요구사항
Frontend: React, TypeScript, Tailwind CSS
API 연동: Google Sheets API v4
상태 관리: React Query 또는 SWR
폼 관리: React Hook Form
UI 컴포넌트: 모던한 컴포넌트 라이브러리
반응형: Mobile-first 접근법
9. 보안 및 성능
API 키 보안: 환경변수를 통한 API 키 관리
데이터 검증: 입력 데이터 유효성 검사
로딩 최적화: 스켈레톤 UI, 지연 로딩
에러 처리: 사용자 친화적 에러 메시지
10. 배포 및 운영

도메인 연결: 기존 도메인 연결 지원
