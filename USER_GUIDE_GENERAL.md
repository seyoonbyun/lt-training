# BNI Korea Leadership Training 2025 - 일반 사용자 가이드

## 🎯 앱 소개

### 특장점 및 기획 의도

**BNI Korea Leadership Training 2025**는 BNI 코리아의 리더십 교육 프로그램을 위한 현대적이고 효율적인 신청 시스템입니다.

#### 🚀 주요 특장점

1. **실시간 연동 시스템**
   - 구글 시트와 실시간 양방향 연동
   - 신청 즉시 관리자 시트에 자동 기록
   - 결제 상태에 따른 실시간 신청자 수 업데이트

2. **스마트 신청 관리**
   - 개별 신청과 일괄 신청 모두 지원
   - 과목별 상태 실시간 반영 (접수중/마감)
   - 결제 완료 기준 정확한 신청자 수 표시

3. **사용자 친화적 인터페이스**
   - 모바일/태블릿/데스크톱 완벽 지원
   - 직관적인 디자인과 간편한 신청 과정
   - BNI 브랜드 컬러 적용된 세련된 디자인

4. **완전 자동화**
   - 수동 데이터 입력 불필요
   - 실시간 강의실 링크 업데이트
   - 자동 신청자 현황 관리

## 📱 메인 화면 안내

### 1. 과목 카드 구성

각 과목 카드에는 다음 정보가 표시됩니다:

- **과목명**: LTT 프로그램 제목
- **일시**: 교육 진행 날짜와 시간
- **장소/강의실**: 
  - 오프라인: 실제 장소 주소
  - 온라인: 강의실 링크 (클릭 시 새 창에서 열림)
- **신청자 수**: "신청자: XX명(결제완료기준)"
- **상태**: 접수중 또는 마감

### 2. 진행 방식 안내 박스

#### 온라인 과목
```
1차: 🖥️ LIVE 진행 (1시간 전 링크 게시)
2차: VOD 녹화본 (익일 오후1시)
※ 온라인 강의실 입장 PW: 신청자 개별 안내
```

#### 파운데이션 T. (오프라인)
```
1차: 오프라인 현장 강의
2차: 🖥️ 온라인 강의실 (익일 오후3시)
※ 온라인 강의실 입장 PW: 신청자 개별 안내
```

## 📝 신청하기

### 개별 신청

1. **과목 선택**
   - 원하는 과목 카드에서 "신청하기" 버튼 클릭
   - 마감된 과목은 버튼이 비활성화됨

2. **신청 유형 선택**
   - "개별 신청" 선택

3. **정보 입력**
   ```
   지역: 드롭다운에서 선택
   챕터명: 텍스트 입력
   성명: 본인 이름
   연락처: 휴대폰 번호 (예: 010-1234-5678)
   이메일: 정확한 이메일 주소
   참여 방식: 실시간 강의 참여 / 녹화본 시청
   특이사항: 문의사항이나 요청사항 (선택사항)
   ```

4. **신청 완료**
   - "신청하기" 버튼 클릭
   - 성공 메시지 확인
   - 구글 시트에 자동 기록됨

### 일괄 신청

1. **과목 선택**
   - 아무 과목에서나 "신청하기" 버튼 클릭
   - "일괄 신청" 선택

2. **템플릿 다운로드**
   - "템플릿 다운로드" 버튼 클릭
   - Excel 파일이 다운로드됨

3. **Excel 파일 작성**
   ```
   A열: 과목명 (예: LTT : PR 코디네이터T.)
   B열: 지역 (예: 서울)
   C열: 챕터 (예: 강남챕터)
   D열: 멤버명 (예: 홍길동)
   E열: 연락처 (예: 010-1234-5678)
   F열: 이메일 (예: hong@example.com)
   G열: 참여 방식 (실시간 강의 참여 또는 녹화본 시청)
   H열: 특이사항 (선택사항)
   ```

4. **파일 업로드**
   - 작성한 Excel 파일을 "파일 선택" 버튼으로 업로드
   - "일괄 신청하기" 버튼 클릭
   - 모든 신청이 한 번에 처리됨

## 💳 결제 안내

### 결제 방법

1. **신청 완료 후**
   - 신청이 완료되면 결제 안내가 표시됨
   - 각 과목 카드에 결제 링크가 제공됨

2. **BNI Korea Store 이용**
   - "결제하기" 링크 클릭
   - BNI Korea Store에서 해당 과목 결제
   - 결제 완료까지 시간이 소요될 수 있음

3. **결제 확인**
   - 결제 완료 후 관리자가 수동으로 확인
   - 확인되면 신청자 수에 반영됨
   - 실시간으로 화면에 업데이트됨

## 🔄 실시간 업데이트

### 자동 새로고침

- **5초마다 자동 업데이트**
  - 신청자 수
  - 과목 상태 (접수중/마감)
  - 강의실 링크

- **즉시 반영 사항**
  - 새로운 신청
  - 결제 상태 변경
  - 관리자의 과목 정보 수정

## 📊 신청 현황 확인

### 대시보드 이용

1. **대시보드 메뉴 클릭**
   - 상단 네비게이션에서 "대시보드" 선택

2. **현황 확인**
   - 지역별 신청 현황
   - 챕터별 신청 현황
   - 과목별 신청 현황
   - 모든 데이터는 결제 완료 기준으로 표시

3. **필터링 기능**
   - 지역, 챕터, 과목별로 필터링 가능
   - 실시간 검색 기능 제공

## ❓ 자주 묻는 질문

### Q1: 신청 후 수정이 가능한가요?
**A**: 현재 시스템에서는 신청 후 수정이 불가능합니다. 수정이 필요한 경우 담당자(joy.byun@bnikorea.com)에게 문의해주세요.

### Q2: 결제 완료 후 언제 반영되나요?
**A**: 결제 완료 후 관리자가 확인하는 대로 실시간 반영됩니다. 보통 영업시간 내에 처리됩니다.

### Q3: 온라인 강의실 비밀번호는 언제 받나요?
**A**: 신청자에게 개별적으로 안내됩니다. 교육 일정이 가까워지면 연락드립니다.

### Q4: 여러 과목을 한 번에 신청할 수 있나요?
**A**: 네, 일괄 신청 기능을 이용하시면 여러 과목을 동시에 신청할 수 있습니다.

### Q5: 마감된 과목도 신청할 수 있나요?
**A**: 마감된 과목은 신청 버튼이 비활성화되어 신청이 불가능합니다.
